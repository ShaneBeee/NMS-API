# NMS-API

NMS-API is a simple library of utility shortcuts for simple NMS things.

## JavaDocs
Check out the [**JavaDocs**](https://shanebeee.github.io/NMS-API/)

## Versions
Currently this library will only work on one version of Minecraft

### Current Version:
- Minecraft 1.20
